Project Idea

A project idea that I have in mind is to construct a robot that will essentially assist in taking care of the environment, minimising the amount of rubbish being littered onto the street. It would mainly do this by just roaming around different areas to collect any litter. Although the main focus is just to clear up all the trash being thrown onto the ground, it could also potentially find any perpetrators and confront them, in hopes of deterring them to continue littering. It would be able to be wirelessly controlled by a human through the a computer or just be able to roam around and detect rubbish through sensors and have a predetermined path programmed to it.

This project would be useful as it would obviously prevent pollution from ending up where it shouldn’t be, such as the oceans or rivers. In 2018, it has now believed that there is approximately 5.25 trillion pieces of plastic remains in the ocean, where 269,000 tons of it has floated onto the surface. Though this is not entirely from litter, it is still a huge portion of it, which could be prevented. Using robots would be a great solution to getting rid of that portion of waste ending up in the oceans, where it wouldn’t affect the environment further as they would be made out of metal. It would be also quite easy to assemble as it would only need to do simple tasks; picking up rubbish and disposing of it properly.

The robot should just be designed to carry out its task in a swift manner with no errors at all times. The structure should also be stable enough so that it wouldn’t be destroyed easily whilst carrying out its purpose. The programming should be sufficient enough so that the robot is aware of its tasks and can carry them out cleanly. If the robot is not being controlled by a human being, it will have a pre-determined path that it will travel on, using a system of motors, drivers and sensors connected together. The sensors will detect any obstacles in the way of the path and would be able to manoeuvre around it with ease. However the a human being would like to control it, they can using a camera to see what the robot can see and wirelessly control it through their computer or other devices.The robot would have a lifting and gripping mechanism, similar to a pair of tongs which would allow it to, of course, pick up the garbage and dispose of it. It would be connected to the motors and also a separate sensor that would detect any rubbish and straight away pick it up. It would also be able to detect the garbage bins to dispose of the rubbish, therefore it would need to be at least a metre in height. It would also need to be at least this height in order for human beings to see it clearly and not damage it by accidentally stepping on it or knocking it over. The design of the robot could be really anything, just as long as it has all these features and is at least one metre tall. However, in the future, if there were to be more reproductions of it, then it should develop a sleeker and visually appealing design so everyone would not mind it in their everyday life. The robot could also be solar powered as it would be outside majority of the time, therefore it would be able to store enough energy through the day. Although it is a great idea, there are many drawbacks after this project such as what if the robot malfunctions and picks up items that are not trash and throws them away, what if someone steals the robots or destroys it, weather conditions it may not work in and many others. In the future, there would probably be some solutions for these, but in the meantime, we would not focus on it yet.

There is quite a lot of software and hardware that is required to construct the robot and program it to achieve its objective. In order for the robot to move, it will need a combination of motors, sensors and drivers that work together to create a pre-determined path. If instead the user wants to control the robot, they can have the option to, using a camera and just having it wirelessly connected to their PC. A gripping and lifting type of mechanism would also need to be created to collect the garbage, that would be connected to some type of motor.

There is quite of lot of skills required for this project to be successful, ranging from programming the robot, to actually physically building it. Any programming language would be useful such as Python, C/C++, Java and others. Robot building skills is also required, whether simple or advanced as the robot doesn’t have to reach a certain point of level yet as it isn’t required to carry out many tasks yet. There would just have to be some assembly of the robot, with building the sensors, wiring the robot together, mounting parts together such as the lifting and gripping mechanism and connecting the motor and wheels.

If the project were to be successful with just one robot, imagine what it would be like with hundreds more around the world working to rid of the litter. This would lead to the decrease of any litter that would end up in the oceans and rivers and just lead to an overall cleaner planet Earth. This development could lead to a bigger project where the robots would be way more sophisticated and have more advanced programming that would allow the robots to so much more than just picking up litter. It could lead to the point where they could make their own decisions and help the world in other ways too.
